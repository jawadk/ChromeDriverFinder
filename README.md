# ChromeDriver Finder

A simple and free web tool to help **QA engineers** quickly find and download the correct **ChromeDriver** for their Chrome version and platform.

---

## Features

- **Auto-detect Chrome version** for your browser  
- **Filter by version and platform** (Windows, Linux, macOS)  
- **Copy download link** or download ChromeDriver directly  
- **About the creator** section with LinkedIn & email for collaborations  
- **PayPal donation button** to support development  
- Easy to use without installing anything  

---

## Why This Tool?

While tools like WebDriver Manager exist, sometimes QA engineers need manual control due to:  

- Legacy projects  
- Organizational security constraints  
- Compatibility testing with specific versions  
- Quick manual lookups without writing scripts  
- Setting up temporary VMs or CI/CD environments  

This tool provides a **fast, web-based solution**.

---

## How to Use

1. Open the [live demo](https://jawadk.github.io/ChromeDriverFinder/ChromeDriverFinder) in your browser.  
2. If your Chrome version is detected, it will pre-select the version.  
3. Choose the **platform** from the dropdown.  
4. Click **Download** or **Copy Link**.
